# Training PPT — Claude Skills Repository

This repository contains Claude skills for working with PowerPoint presentations.

---

## How to Install a Skill

Run the following command in your terminal, replacing the URL with the skill you want:

```bash
claude skill install https://github.com/Eliyahubi/Training_PPT/raw/main/skills/<skill-name>/SKILL.md
```

### Available Skills

| Skill | Description | Install Command |
|-------|-------------|-----------------|
| `pptx-annotate` | Adds numbered annotation circles on screenshots inside PPTX slides, fixes Hebrew titles, and fixes label box layout | `claude skill install https://github.com/Eliyahubi/Training_PPT/raw/main/skills/pptx-annotate/SKILL.md` |

---

## After Installing

Once installed, Claude will automatically recognize when to use the skill based on your requests. You can also trigger it manually by describing what you need, for example:

- *"Add numbered markers to the screenshots in this presentation"*
- *"הוסף מספרים לתמונות בפרזנטציה"*

הרץ את הפקודה הבאה בטרמינל, עם הסקיל שאתה רוצה:


| סקיל | תיאור | פקודת התקנה |
|------|-------|-------------|
